# Raw Billing Evidence

This folder contains raw-style supporting documents for billing validation. The files intentionally avoid agent-ready checklists and explicit billing decisions inside the evidence.

| Billing Type | Evidence Style |
|---|---|
| Managed Services | Email messages with SLA/KPI target and actual values |
| ERP Based | Plain Word hypercare notes with target and actual performance |
| T&M | Excel timesheets with daily employee hours |
| T&M Over Cap | Excel cap tracker extracts with utilization and approval references |
| Outcome Based | Customer email approval / hold threads |
| Milestone Based | Customer milestone acceptance / non-acceptance email threads |
| Device Based | Excel active-device inventory snapshots |
| ARC | Excel ARC consumption ledgers |
| RRC | Excel renewal run commitment ledgers |

There are two samples for each billing type so agents can test both straightforward and problematic billing cases by reading the source evidence.
